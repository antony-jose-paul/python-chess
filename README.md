# Python Chess ♟️

A terminal-based chess game built in Python, developed step by step with a focus on **core game logic, clean design, and learning by building**.

This project implements the full rules of chess and is designed to be extended later with a GUI or AI.

---

## Features
- Standard 8×8 chess board
- Turn-based gameplay
- Legal move validation for all pieces
- Check, checkmate, and stalemate detection
- Special moves:
  - Castling
  - Pawn promotion
  - En passant (optional / planned)
- Move history and undo support
- Clean, modular code structure
- Terminal-based user interface

---

## Tech Stack
- **Language:** Python 3
- **Interface:** Terminal (ASCII-based board)
- **Version Control:** Git & GitHub
